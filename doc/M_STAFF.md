# 社員マスタ一覧（m_staff）

---

## 1. 概要

社員マスタの一覧表示・検索・ソート・ページング・新規登録遷移・詳細遷移・CSVエクスポートを行う画面。

本画面では以下を実現する。

* 社員情報の検索・絞り込み
* 一覧表示
* 詳細画面への遷移
* 新規登録画面への遷移
* CSVエクスポート
* 一覧状態の保持

---

## 2. 対応ファイル

| 種別    | ファイル名              |
| ----- | ------------------ |
| 画面    | m_staff.cfm        |
| API   | m_staff.cfc        |
| JS    | m_staff.js         |
| CSV出力 | m_staff_export.cfm |

---

## 3. 機能一覧

### 3.1 検索機能

以下の条件で検索可能。

* 社員コード
* 社員名
* メールアドレス
* 権限区分
* 使用区分

---

### 3.2 Enterキーによるフォーカス移動

* Enter押下で次の検索項目へ移動
* IME変換中は無視
* Enterで検索は実行しない

---

### 3.3 一覧表示

表示項目：

* 社員コード
* 社員名
* 権限区分
* メールアドレス
* 使用区分
* 作成日時
* 更新日時

---

### 3.4 権限区分表示

| 値   | 表示  |
| --- | --- |
| 9   | 管理者 |
| 2   | 責任者 |
| その他 | 一般  |

バッジ表示で区別する。

---

### 3.5 使用区分表示

| 値   | 表示  |
| --- | --- |
| 1   | 使用中 |
| その他 | 停止  |

---

### 3.6 行クリック → 詳細画面遷移

* 行クリックで詳細画面へ遷移（POST）
* 表示モード：view

保持される内容：

* 検索条件
* ソート条件
* ページ番号

---

### 3.7 新規登録

* ヘッダの「追加」ボタン押下
* display_mode = add で詳細画面へ遷移

---

### 3.8 ソート機能

状態遷移：

未指定 → 昇順 → 降順 → 未指定

対象項目：

* 社員コード
* 社員名
* 権限区分
* メールアドレス
* 使用区分
* 作成日時
* 更新日時

---

### 3.9 ページング

* 1ページ50件
* 操作ボタン

  * 最初
  * 前へ
  * 次へ
  * 最後

---

### 3.10 CSVエクスポート

検索条件・ソート条件を保持したままCSV出力。

```javascript
window.location.href = "m_staff_export.cfm?" + params.toString();
```

## 4. API仕様

### 4.1 一覧取得API

m_staff.cfc?method=getStaffList

#### リクエスト

```json
{
  "page": 1,
  "pageSize": 50,
  "search_staff_code": "",
  "search_staff_name": "",
  "search_mail_address": "",
  "search_authority_level": "",
  "search_use_flag": "",
  "sortField": "",
  "sortOrder": ""
}
```

#### レスポンス

```json
{
  "status": 0,
  "results": [],
  "paging": {
    "page": 1,
    "totalPage": 10,
    "totalCount": 100
  }
}
```

---

### 4.2 ステータス仕様

| status | 意味      |
| ------ | ------- |
| 0      | 正常      |
| 1      | 異常（エラー） |

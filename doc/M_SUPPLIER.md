# 取引先マスタ一覧（m_supplier）

---

## 1. 概要

取引先マスタの一覧表示・検索・ソート・ページング・新規登録遷移・詳細遷移・CSVエクスポートを行う画面。

本画面では以下を実現する。

* 取引先情報の検索・絞り込み
* 一覧表示
* 詳細画面への遷移
* 新規登録画面への遷移
* CSVエクスポート
* 一覧状態の保持

---

## 2. 対応ファイル

| 種別    | ファイル名                 |
| ----- | --------------------- |
| 画面    | m_supplier.cfm        |
| API   | m_supplier.cfc        |
| JS    | m_supplier.js         |
| CSV出力 | m_supplier_export.cfm |

---

## 3. 機能一覧

### 3.1 検索機能

以下の条件で検索可能。

* 取引先コード
* 取引先名
* 配送業者
* 使用区分（有効 / 無効）

---

### 3.2 Enterキーによるフォーカス移動

* Enter押下で次の検索項目へ移動
* IME変換中は無視
* Enterで検索は実行しない

---

### 3.3 一覧表示

表示項目：

* 取引先コード
* 取引先名
* 配送業者
* 使用区分
* 作成日時
* 更新日時

---

### 3.4 使用区分表示

| 値   | 表示 |
| --- | -- |
| 1   | 有効 |
| その他 | 無効 |

バッジ表示で視覚的に判別する。

---

### 3.5 行クリック → 詳細画面遷移

* 行クリックで詳細画面へ遷移（POST）
* 表示モード：view

保持される内容：

* 検索条件
* ソート条件
* ページ番号

---

### 3.6 新規登録

* ヘッダの「追加」ボタン押下
* display_mode = add で詳細画面へ遷移

---

### 3.7 ソート機能

状態遷移：

未指定 → 昇順 → 降順 → 未指定

対象項目：

* 取引先コード
* 取引先名
* 配送業者
* 使用区分
* 作成日時
* 更新日時

---

### 3.8 ページング

* 1ページ50件
* 操作ボタン

  * 最初
  * 前へ
  * 次へ
  * 最後

---

### 3.9 CSVエクスポート

検索条件・ソート条件を保持したままCSV出力。

```javascript
window.location.href = "m_supplier_export.cfm?" + params.toString();
```

## 4. API仕様

### 4.1 一覧取得API

m_supplier.cfc?method=getSupplierList

#### リクエスト

```json
{
  "page": 1,
  "pageSize": 50,
  "search_supplier_code": "",
  "search_supplier_name": "",
  "search_delivery_company": "",
  "search_use_flag": "",
  "sortField": "",
  "sortOrder": ""
}
```

#### レスポンス

```json
{
  "status": 0,
  "results": [],
  "paging": {
    "page": 1,
    "totalPage": 10,
    "totalCount": 100
  }
}
```

---

### 4.2 ステータス仕様

| status | 意味      |
| ------ | ------- |
| 0      | 正常      |
| 1      | 異常（エラー） |

---

## 5. 検索仕様

### 5.1 取引先コード

* 完全一致

---

### 5.2 取引先名

* 部分一致検索

---

### 5.3 配送業者

* 完全一致または選択式

---

### 5.4 使用区分

| 値   | 条件   |

| 0   | 無効のみ |

## 6. ソート仕様

| sortField        | 対象     |
| ---------------- | ------ |
| supplier_code    | 取引先コード |
| supplier_name    | 取引先名   |
| delivery_company | 配送業者   |
| use_flag         | 使用区分   |
| create_datetime  | 作成日時   |
| update_datetime  | 更新日時   |

## 11. エラーハンドリング

---

## 12. 注意点

* status = 0 が正常
* Enterで検索実行しない
* IME考慮あり
* ソート3段階制御
* 状態保持あり

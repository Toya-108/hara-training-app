# 伝票管理機能（slip）

---

## 1. 概要

本機能は、発注伝票の登録・確認・確定・管理を行うための機能である。

主に以下の業務を対象とする：

* 発注伝票の作成
* 伝票の一覧確認
* 伝票内容の修正
* 伝票の確定処理
* データの集計・出力

---

## 2. 対応画面

| 画面名  | ファイル             | 説明         |
| ---- | ---------------- | ---------- |
| 伝票一覧 | slip_list.cfm    | 伝票の検索・一覧表示 |
| 伝票詳細 | slip_list_dt.cfm | 伝票の詳細表示・編集 |

---

## 3. 画面構成

```
伝票一覧 → 伝票詳細 → （編集）→ 保存 → 一覧へ戻る
```

---

## 4. 業務フロー

### 4.1 基本フロー

1. 伝票を検索する
2. 一覧から対象伝票を選択
3. 詳細画面で内容を確認
4. 必要に応じて編集
5. 保存または確定

---

### 4.2 一括確定フロー

1. 一覧画面で検索
2. 対象伝票を絞り込み
3. 一括確定ボタン押下
4. パスワード入力
5. 確認後、確定処理実行

---

## 5. ステータス管理

| ステータス | 値 | 説明        |
| ----- | - | --------- |
| 登録    | 1 | 作成済み（未確定） |
| 確定    | 2 | 確定済み      |
| 削除    | 3 | 削除済み      |

---

## 6. 主要機能

### 6.1 一覧機能

* 検索（複数条件）
* ソート（昇順 / 降順）
* ページング
* CSVエクスポート
* 一括確定

---

### 6.2 詳細機能

* 伝票ヘッダ表示
* 明細表示
* 編集モード切替
* 商品コード補完
* 数量・単価入力
* 自動金額計算
* 保存処理

---

## 7. データ構成

### 7.1 ヘッダ（t_slip）

* 伝票番号
* 発注日
* 納品日
* 取引先
* ステータス
* 合計数量
* 合計金額

---

### 7.2 明細（t_slip_detail）

* 商品コード
* JANコード
* 商品名
* 数量
* 単価
* 金額

---

## 8. 状態管理（重要）

一覧 → 詳細 → 戻る際に以下を保持：

* 検索条件
* ページ番号
* ソート条件

POST + hidden項目で管理する。

---

## 9. 技術仕様

* フロント：JavaScript（async / await）
* UI制御：SweetAlert2
* 日付入力：flatpickr
* API：CFML（CFC）
* データ形式：JSON

---

## 10. 設計方針

* 画面単位で責務を分離（cfm / cfc / js）
* 状態保持を重視（UX向上）
* APIは明確なJSON構造
* エラーハンドリングは画面側で統一
